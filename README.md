# きみどりマップ
微地形図と林相識別図を表示する林業現場用の地図です。

![rrim](https://github.com/user-attachments/assets/59f859f1-bb87-413c-a06e-2c2529a20fd7)

## 特徴
- 微地形図と林相識別図（オープンデータ化されたもののみ）を表示する地図です。
- 林業の現場作業用です。そのため、スマートフォンでの利用を想定しています。
- 通信圏外で利用が可能です。ただし、プログラムと地図を事前に保存する必要があります。

## 機能
- 現在地の表示
- 地図の表示
- 地図の端末への保存
- 地点登録
- 外部のGISデータのインポート（座標系はWGS84のみ）。
  
## オフラインでの使い方
- アプリのインストールと地図の保存が必要です。
  - ブラウザのメニューから「ホームに追加」を押してインストールします。
  - 保存したい地図を、保存したいエリアで表示し、画面右上の保存ボタンを押します。
- 注意点として、地図を保存するときは、必要最小限のエリアに絞ってください。地図を配信するサーバーの負荷を軽減するためです。
